HTTP and AJAX

This topic covers using the XMLHttpRequest object to perform AJAX requests. For more information, please refer to the MDN Using XMLHttpRequest documentation.

At the end of this topic, you should be able to...

  • Define the acronyms HTTP, HTML, AJAX, XML, and JSON
  • List the four phases of an HTTP Process
    • Connection
    • Request
    • Response
    • Disconnection
  • List the five common request methods that can be placed in the HTTP Request phase
    • HEAD
    • GET
    • POST
    • PUT
    • DELETE
  • Describe what is meant by the statement that the HTTP process is "stateless"?
  • List some of the common status codes returned by the web server.
  • Distinguish between synchronous and asynchronous operations.
  • Describe the following methods of the XMLHttpRequest object: open, setRequestHeader, send, getAllResponseHeaders, getResponseHeader, abort
  • Describe the following properties of the XMLHttpRequest object: onload, onreadystatechange, readyState, responseText, responseXML, status, statusText
  • Describe the following events of the XMLHttpRequest object: loadStart, progress, abort, error, load, timeout, loadend, readystatechange
  • Use the XMLHttpRequest object to load JSON data
