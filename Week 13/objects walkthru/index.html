Objects Walk Through

At the end of this topic, you should be able to...

  • Identify similarities and differences between objects in statically types languages (e.g.: C#) and JavaScript.
  • Explain what is mean by "Prototypical Inheritance".
  • Use the .prototype property to extend objects.
  • Use the .hasOwnProperty() method to identify if a property exists on an object
  • Explain how "encapsulation" is achieved in JavaScript objects.
  • Use a constructor to instantiate a custom object.
  • Create an object as an "object literal".
  • Define the term JSON
  • Identify the characteristics of JSON objects.
  • Translate JSON objects to and from strings.
